# SDSU_SCIL_CensusData_AI_Assistant-
Gemini powered AI assistant to allow clean mapping of census data. 

The Downloader notebook allows for downloading Census dating using an APi key, however the resulting parquet file can be downloaded directly from the github repo  
  
The viewer shows the setup and several examples for using the FEMA/Census data viewer, however a Gemini API key is needed for functionality.  

Examples folder shows example dashboard output in HTML form
